# Conselhos Administrativo - Frontend

Este é o diretório contendo a aplicação frontend do sistema de Gestão de Conselhos.

Para informações sobre arquitetura geral, stack (React + Express), e instruções de execução de todo o sistema, consulte o **[README principal do projeto](../README.md)** na raiz do repositório.

## Scripts Disponíveis

- `npm run dev`: Inicia o servidor de desenvolvimento do Vite.
- `npm run build`: Compila o projeto TypeScript e realiza o build de produção via Vite.
- `npm run lint`: Executa a verificação do ESLint no código.
- `npm run preview`: Inicia um servidor local para visualizar o build de produção.
